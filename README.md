# AI Slide Generator - Презентацияларды AI менен түзүү платформасы

Бул долбоор колдонуучуларга каалаган темада Google Gemini AI жардамы менен профессионалдуу презентацияларды секунда ичинде түзүүгө мүмкүнчүлүк берет.

## 🚀 Негизги функциялары

-   **AI Генерация:** Тема боюнча слайддардын мазмунун жана түзүмүн автоматтык түрдө түзүү (Gemini 1.5 Pro).
-   **Интерактивдүү Редактор:** Түзүлгөн слайддарды онлайн режиминде түзөтүү жана сактоо.
-   **Экспорттоо:** Презентацияларды **PowerPoint (.pptx)** жана **PDF** форматтарында жүктөп алуу.
-   **Бөлүшүү:** Презентацияларды ачык шилтеме аркылуу коомчулукка көрсөтүү.
-   **Админ Панель:** API токендерин башкаруу жана колдонуучулардын төлөмдөрүн тастыктоо.
-   **Төлөм Системасы:** QR-код же карта аркылуу жазылууну сатып алуу (Premium мүмкүнчүлүктөр).

## 🛠 Технологиялык стек

-   **Frontend:** Next.js 14+ (App Router), Tailwind CSS, Shadcn/UI.
-   **Backend:** Next.js Server Actions.
-   **Database & Auth:** Supabase (PostgreSQL & Google Auth).
-   **AI Integration:** Google Generative AI (Gemini API).
-   **Export Tools:** PptxGenJS, html2canvas, jsPDF.

## ⚙️ Орнотуу жана иштетүү

1.  **Репозиторийди көчүрүү:**
    ```bash
    git clone <repository-url>
    cd aislaide/ai-slide-generator
    ```

2.  **Керектүү пакеттерди орнотуу:**
    ```bash
    npm install
    ```

3.  **Айлана-чөйрө өзгөрмөлөрүн жөндөө (.env.local):**
    Керектүү өзгөрмөлөрдү (Supabase URL, Anon Key, Gemini API Key ж.б.) `.env.local` файлына кошуңуз.

4.  **Долбоорду иштетүү:**
    ```bash
    npm run dev
    ```
    Тиркеме `http://localhost:3000` дарегинде ачылат.
