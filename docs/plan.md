🚀 AI Slide Generator - Долбоордун Техникалык Планы

Бул документ презентацияларды AI жардамы менен түзүүчү платформаны иштеп чыгуу үчүн негизги колдонмо болуп саналат.

### Долбоордун Техникалык Тапшырмалары (ТЖ)

Бул план үч негизги бөлүктөн турат:
1. **[ТЖ 1: Базалык Структура](./base_structure.md)** — Архитектура, Тех. стек жана Маалымат базасы.
2. **[ТЖ 2: Админ Панель](./admin_panel.md)** — Настройкалар, Токендер жана Төлөмдөрдү тастыктоо.
3. **[ТЖ 3: Колдонуучу Баракчасы](./user_dashboard.md)** — Тарифтер, Төлөмдөр жана Билдирүүлөр тарыхы.

### 1. Технологиялык Стек
- **Framework:** Next.js 14+ (App Router).
- **AI:** Google Gemini 1.5 Pro/Flash.
- **Database:** Supabase (Auth & PostgreSQL).
- **Styling:** Tailwind CSS + Shadcn/UI (Ак фондо, заманбап дизайн).
- **Export:** PptxGenJS (PowerPoint) / Puppeteer (PDF).

### 2. Иштеп Чыгуу Кадамдары (Roadmap)

#### 1-этап: Инфраструктура жана Авторизация
- [x] `docs` папкасынын ичинде Next.js долбоорун баштоо жана Tailwind/Shadcn UI орнотуу.
- [x] Supabase Auth (Google Login) интеграциясы.
- [x] Маалымат базасынын схемасын түзүү (`users`, `subscriptions`, `notifications`).
- [x] Middleware аркылуу `/admin`, `/dashboard` жана `/edit` баракчаларын ролдорго (RBAC: ADMIN, TEACHER, USER) ылайык коргоо.
- [x] Базалык баракчаларды (`/dashboard`, `/admin`, `/editor`) түзүү жана навигацияны кошуу.
- [x] `EXTERNAL_CHECK_USER_URL` аркылуу тышкы роль текшерүү логикасын кошуу.
