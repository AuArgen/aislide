# 📄 ТЖ 2: Админ Панель жана Башкаруу

Бул документ администраторлор үчүн башкаруу куралдарын жана функционалдык талаптарды аныктайт.

### 1. Настройкалар (Settings Manager)
- **Токендер жана Ключтор:** Gemini API Key, Unsplash API Key жана башка үчүнчү тараптын ключтарын админ панелден кошуу жана өзгөртүү мүмкүнчүлүгү.
- **Төлөм Настройкасы:** Админдин картасынын QR-кодун (сүрөт катары) жүктөө жана карта номерин жазуу. Бул маалымат колдонуучуга төлөм учурунда көрсөтүлөт.

### 2. Колдонуучуларды Көзөмөлдөө
- **Колдонуучулардын Тизмеси:** Бардык катталган колдонуучуларды издөө жана чыпкалоо (фильтр).
- **Доступ Башкаруу:** Колдонуучунун статусун же ролун (User/Teacher/Admin) өзгөртүү мүмкүнчүлүгү.
- **Акча Төлөгөндөр:** Төлөм кылган колдонуучуларга "Teacher" (Premium) статус берүү же убактысын (expires_at) узартуу.

### 3. Төлөмдөрдү Тастыктоо (Payment Verification)
- **Сурамдардын Тизмеси:** Колдонуучулар жөнөткөн төлөм сурамдарын (чектин сүрөтү же PDF) көрүү.
- **Чечим Кабыл Алуу:** "Подтвердить" (Тастыктоо) же "Отменить" (Баш тартуу) баскычтары.
- **Автоматташтыруу:** Админ төлөмдү тастыктаганда, колдонуучунун подписка статусу автоматтык түрдө "Active" болуп өзгөрүшү керек.

### 4. Билдирүүлөр жана Кабарлашуу (Notifications)
- **Telegram Интеграциясы:** Жаңы төлөм сурамы түшкөндө, админдин Телеграм ботуна же группасына дароо маалымат барышы керек.
- **Колдонуучуга Жооп:** Админ чечим кабыл алганда (Подтвердить/Отменить), колдонуучунун жеке кабинетине билдирүү (статус) жөнөтүү.

### 5. Дизайн жана UI
- **Интерфейс:** Shadcn/UI компоненттерин колдонуу менен таза жана түшүнүктүү таблицалар жана формалар.
- **Түстөр:** Ак фондо, маанилүү баскычтар гана ачык түстө.
